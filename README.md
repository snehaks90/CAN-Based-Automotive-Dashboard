# Automotive Dashboard System using CAN Protocol

## Overview

This project is an Automotive Dashboard System developed using PIC18F4580 microcontrollers and CAN protocol.

The system consists of three ECUs communicating through a CAN bus network to exchange vehicle information in real time.

## Features

- Multi-node CAN communication
- Transmission of vehicle speed
- RPM monitoring
- Gear position indication
- Turn indicator status transmission
- Real-time dashboard display on CLCD
- Automotive ECU simulation

## Hardware Used

- PIC18F4580 Microcontroller
- MCP2551 CAN Transceiver
- Character LCD (CLCD)
- Push Buttons / Switches
- CAN Bus Network

## Software Used

- Embedded C
- MPLAB IDE
- XC8 Compiler

## Working

1. Vehicle parameters are generated by the transmitting ECU.
2. Data is packed into CAN frames.
3. CAN messages are transmitted over the CAN bus.
4. Receiving ECU extracts the data.
5. Vehicle information is displayed on the CLCD.
